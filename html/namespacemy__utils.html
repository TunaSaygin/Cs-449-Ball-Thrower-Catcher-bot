def my_utils.find_line_intersections_2d (   bin_pos,
  initial_position,
  bin_dimensions 
)

Finds the intersection points of a line with the boundaries of a rectangular bin in the x-y plane.

The line is defined by its start position (initial_position) and a target position (bin_pos). The bin boundaries are defined by a dictionary containing the bottom-left and top-right coordinates.

Parameters
bin_posPosition of the bin center [x, y].
initial_positionInitial position of the line [x, y].
bin_dimensionsDictionary with bin dimensions:
  • 'bottom_left': [x_min, y_min]
  • 'top_right': [x_max, y_max].
Returns
list A list of intersection points [x, y] with the bin boundaries.
